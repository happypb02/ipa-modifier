# 快速开始指南 - 使用 GitHub Actions 修改 IPA

## 第一步: 创建 GitHub 仓库

1. 访问 https://github.com/new
2. 仓库名称: `ipa-modifier` (或任意名称)
3. 选择 **Private** (私有仓库,保护你的 IPA 文件)
4. 点击 "Create repository"

## 第二步: 上传文件

### 方法 A: 使用 Git 命令行 (推荐)

在当前目录 (Desktop) 打开终端,运行:

```bash
# 初始化 Git 仓库
git init

# 添加文件
git add .github/ InstallRedirect/ 2.ipa

# 提交
git commit -m "Initial commit: Add IPA modifier"

# 连接到 GitHub (替换成你的用户名和仓库名)
git remote add origin https://github.com/你的用户名/ipa-modifier.git

# 推送
git branch -M main
git push -u origin main
```

### 方法 B: 使用 GitHub 网页上传

1. 进入你创建的仓库
2. 点击 "uploading an existing file"
3. 拖拽以下文件夹和文件:
   - `.github` 文件夹
   - `InstallRedirect` 文件夹  
   - `2.ipa` 文件
4. 点击 "Commit changes"

## 第三步: 等待自动构建

上传完成后:

1. 点击仓库顶部的 **"Actions"** 标签
2. 你会看到 "Build Modified IPA" 工作流正在运行
3. 等待 5-10 分钟 (第一次会慢一些)
4. 构建完成后会显示绿色的 ✓

## 第四步: 下载修改后的 IPA

构建成功后:

1. 点击最新的构建记录
2. 滚动到页面底部,找到 **"Artifacts"** 区域
3. 点击 **"modified-ipa"** 下载
4. 解压 zip 文件,得到 `2_modified.ipa`

## 第五步: 安装到设备

修改后的 IPA 需要签名才能安装。推荐使用:

### 使用 AltStore (Windows 可用,免费)

1. 下载 AltStore: https://altstore.io/
2. 安装 AltServer 到电脑
3. 连接 iOS 设备到电脑
4. 在 AltStore 中选择 "+" 添加 IPA
5. 选择 `2_modified.ipa` 并安装

### 使用 Sideloadly (Windows 可用,免费)

1. 下载: https://sideloadly.io/
2. 连接 iOS 设备
3. 拖拽 `2_modified.ipa` 到 Sideloadly
4. 输入 Apple ID 并点击 "Start"

### 使用爱思助手 (Windows 可用,免费)

1. 下载爱思助手
2. 连接设备
3. 点击 "应用游戏" → "导入安装"
4. 选择 `2_modified.ipa`

## 验证修改是否生效

安装后,打开应用:

1. 进入 "安装包定制" 功能
2. 定制一个应用
3. 点击 "去安装" 按钮
4. **应该会跳转到 "文件-已定制" 页面** (而不是原来的跳转)

## 故障排除

### 构建失败

查看 Actions 页面的日志:
- 点击失败的构建
- 查看红色 ✗ 的步骤
- 复制错误信息告诉我

### 无法安装 IPA

- **"无法验证应用"**: 需要在设置中信任开发者证书
- **"证书已过期"**: 免费证书 7 天过期,需要重新签名
- **"应用已损坏"**: 下载可能不完整,重新下载

## 下次修改

如果需要调整跳转逻辑:

1. 修改 `InstallRedirect/Tweak.x` 文件
2. 推送到 GitHub: `git add . && git commit -m "Update" && git push`
3. GitHub Actions 会自动重新构建
4. 下载新的 IPA
